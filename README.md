# Prosjektoppgave2IT
Jeg er fortsatt litt usikker over hva prosjektet mitt skal være, men den første ideen skulle være en nettside av Kuben kantine der andre som ikke vet hva som skal være i katina kan endelig få vite hva som er i kantina gjennom nettsiden mitt. Jeg er litt inspirert av nettbutikk forslaget så jeg gjetter jeg kommer til å bruke databaser og python for at dette skulle fungere.
Kodespråket jeg skal bruke skal inneholde Python, HTML, CSS og Javascript.
Jeg kommer til å kode masse som HTML, CSS og JAVA dette kan gi meg høyere kompetanse i Utvikling, etter det kommer jeg til å hente produktene mine fra databaser som er hentet fra python dette kan bidra for Drift og Brukerstøtte kompetanse
https://www.figma.com/design/Zg4kRPGNz6su7xyzooYSyV/KubenKantina?node-id=0-1&p=f&t=D5p1lGQXEDTW9i2E-0

Jeg har forandret ideen mitt til et nettbutikk som handler om at andre kan komme inn på nettsiden og finne Manga produkter der. Det er samme kodespråkene som den forrige ideen
UKE 1:
Jeg hadde begynt med å lage en skisse over nettsiden min/frontend: https://www.figma.com/design/Zg4kRPGNz6su7xyzooYSyV/MANGAJUMP?node-id=0-1&p=f&t=uQ0pTU3glkqsKp3f-0
Etter jeg hadde blitt ferdig med skissen min så started jeg koden, først begynte jeg med hovedsiden som tok sitt tid pågrunn av problemer av navbar og bilde men det klarte jeg å bli ferdig med i UKE 1
UKE 2:
Jeg fortsettet å jobbe med nettsiden der jeg skulle legge inn produktene mine, dette tok lengre tid enn forventet ikke pågrunn av HTML OG CSS men pågrunn av java. Jeg fant ikke en vei til å lage en kode som gjør at hvis du klikker på navbaren 'manga' så skulle det skinne rødt imens det drar deg ned til siden Manga. Dette førte at jeg ditcha å lage en kode i js men å lage et script i HTML, dette var et bra løsning pågrunn at den gjorde alt det jeg egentlig ville som jeg ikke klarte i JS. Da jeg klarte å fullføre problemet så fortsettet jeg med HTML/CSS og legge inn bilder av produktet. Jeg også startet litt på backend og sette inn mariadb i Ubuntu men det klarte ikke så jeg ga opp litt med det for nå.
UKE 3:
Jeg fortsettet ferdig Manga siden mitt da fortsettet jeg med Produkt siden mitt der alle merchandise for typ One piece toys kunne ligge som var enkelt, alt jeg gjorde var å kopiere den forrige siden til denne og forandre bildene. Tilslutt så lagde jeg en footer der mine sosiale medier og github kunne ligge.
Når jeg ble ferdig med frontend så gikk jeg til backend, først så startet jeg opp mariadb i ubuntu som jeg klarte å gjøre ved å bruke mariadb -u root -p som gjør at du skal skrive username: -u og passord: -p.
